콘솔로 이동(Window 기준)    
  
1. gradlew build -> build 폴더 생성  
2. cd build/libs -> hello-spring-0.0.1-SNAPSHOT.jar 파일이 생성됨
3. java -jar hello-spring-0.0.1-SNAPSHOT.jar -> 서버 실행 시작.
4. 실행 확인
  
- ^c : 서버 종료
- /study/hello-spring : 
  - gradlew clean -> build 폴더 삭제. 
  - gradlew clean build : .jar 파일 지우고 다시 생성.
